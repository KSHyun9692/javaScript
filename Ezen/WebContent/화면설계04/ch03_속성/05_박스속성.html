박스속성

종류

width 폭(넓이)
height 높이
padding 안쪽여백
border 테두리
margin 주변여백

width, height

크기단위 입력(px, %, em, vw,vh)

padding

padding-top, padding-right, padding-botton, paddin-left
ex) padding: 10px; 위쪽으로 10px 오른쪽으로 10px 아래쪽 10px 왼쪽 10px padding: 0 10px; 상하 0px 우좌 10px; padding: 0 10px 20px; 상0px, 우좌 10px 하 20px padding: 0 10px 20px 30px; 상0px 우10px 하20px 좌30px

margin 속성

padding과 입력방법은 같음

border 속성 : 3가지 속성이 모두 입력되어애 정확히 출력됨

border-width 두께(크기단위 입력)
border-color 색상(색상단위 입력)
border-style 테두리 스타일

주로 한번에 입력함 border: width style color; 방향별 border 성정시 border-top,right,bottom, left

border-style

none 없음 solid 실선 dashed 대시선 dotted 점선 double 이중선 outset 양각선 inset 음각선 ridge 테두리만 양각선 groove 테두리만 음각선

border-radius

박스모델 모서리부분을 둥글게 만들때
border-radius: 좌상 우상 우하 좌하; 왼쪽 상단기준으로 4개 값이 시계방향으로 적용
동일한 값을 입력할 때는 값을 하나만 입력하면 됨.
